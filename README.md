# PythonProject
 
## Pierwsze uruchomienie
 
Linux/macos
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
 
Windows
```powershell
py -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```
